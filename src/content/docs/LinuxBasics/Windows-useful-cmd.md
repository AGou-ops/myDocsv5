# windows 命令指示符

- `ipconfig /flushdns`：刷新DNS缓存；
- `netstat -abn`：查看当前所有连接；
- `netsh interface portproxy add v4tov4 listenaddress=127.0.0.1 listenport=9000 connectaddress=192.168.0.10 connectport=80`：添加端口映射；
